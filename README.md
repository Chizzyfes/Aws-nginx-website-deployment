What is Freshdesk**

Freshdesk is a cloud based customer support and helpdesk platform developed by Freshworks. It helps businesses manage customer conversations from different channels in one centralized system. With Freshdesk, support teams can respond faster, track issues easily, and improve customer satisfaction.

Freshdesk is commonly used by startups, small businesses, and large organizations to handle customer queries professionally.

What Freshdesk Is Used For**

Freshdesk is mainly used to:

- Manage customer support tickets
- Respond to customer issues from multiple channels
- Track agent performance
- Automate repetitive support tasks
- Build a self-service help center

Freshdesk Dashboard Overview**

The Freshdesk dashboard is the main workspace where agents and admins monitor all support activities.

From the dashboard, you can:

- View open, pending, resolved, and closed tickets
- Monitor agent workload
- Track customer response time
- Access reports and analytics

The dashboard gives a quick overview of how the support team is performing.

Tickets in Freshdesk**

A ticket is created whenever a customer contacts support.

**Sources of Tickets**

Tickets can come from:

- Email
- Live chat
- Phone calls
- Social media (Facebook, Twitter/X)
- Web forms
- WhatsApp (with integration)

Each ticket contains:

- Customer details
- Issue description
- Priority level
- Status
- Assigned agent


## Conclusion

This project demonstrates the successful deployment of a secure static portfolio website on AWS EC2 using Nginx as the web server.

The implementation covered real-world cloud engineering practices including server provisioning, SSH security hardening, Elastic IP configuration, domain DNS integration, HTTPS SSL certificate installation, and infrastructure monitoring using Datadog.

Through this project, practical experience was gained in Linux server administration, cloud networking, web server configuration, and production deployment workflows.

The result is a live, secure, and monitored web application accessible via a custom domain.
